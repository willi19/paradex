class paradex.io.robot_controller.under_test.hand_rh56dftp.InspireHandRH56DFTP(port: str = '/dev/ttyUSB0', baudrate: int = 115200, slave_id: int = 1, debug: bool = False)¶

Interface for controlling the Inspire Hand RH56DFTP.

This class provides functionality for: - Connection management - Reading current pose - Writing/executing pose - Reading contact information

modbus¶
connect()¶

Context manager for connecting to the hand.

Example

with hand.connect():

pose = hand.read_pose() hand.write_pose([1000, 1000, 1000, 1000, 1000, 1000])

open() → None¶

Connect to the hand.

Raises:

ConnectionError – If connection fails

close() → None¶

Disconnect from the hand.

property is_connected: bool¶

Check if the hand is connected.

Read the current pose of all joints.

Returns:

List of current pose values for all joints (typically 6 values for 6 DOF)

Raises:

  • ConnectionError – If not connected

  • CommandError – If command fails

write_pose_by_id(id: int, pose: int) → None¶

Write/execute a pose for all joints.

Parameters:

  • id – Joint ID (0-5)

  • pose – Pose value for the joint (0-2000, -1 for no action)

  • DOF (order of) – lf, rf, mf, if, thumb bending, thumb rotating

  • 0 – open 2000: closed

Raises:

  • ConnectionError – If not connected

  • CommandError – If command fails

  • ValueError – If pose values are invalid

Set the force threshold for all joints.
