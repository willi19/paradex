.. py:function:: triangulation(corners: numpy.ndarray, projections: numpy.ndarray, threshold=1.5, iterations=100)

   corners : Nx2 matrix (N cameras, 2D image coordinates)

   iterations : Number of RANSAC iterations

   :returns: (4, 3) matrix with filtered 3D keypoints
