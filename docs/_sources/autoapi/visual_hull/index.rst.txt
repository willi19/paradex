.. py:function:: plot_3d_points(points, title='3D Points', color='blue', size=20, alpha=0.6, subsample=None, show_axes=True, figsize=(10, 8))

   3D points를 간단하게 시각화하는 함수

   :param points: Nx3 array of 3D points
   :param title: 제목
   :param color: 점 색상 ('blue', 'red', 'green' 등 또는 RGB tuple)
   :param size: 점 크기
   :param alpha: 투명도 (0-1)
   :param subsample: 너무 많으면 N개만 랜덤 샘플링 (None이면 모든 점 표시)
   :param show_axes: 축 표시 여부
   :param figsize: figure 크기
