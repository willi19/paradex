.. py:function:: compute_mesh_to_ground_distance(obj_pose, mesh)

   Compute the minimum distance between the mesh's lowest point and the ground (z=0).

   :param obj_traj: Object trajectory where each element contains
                    the position (x, y, z) of the object.
   :type obj_traj: list or np.ndarray
   :param mesh: Mesh representing the object.
   :type mesh: open3d.geometry.TriangleMesh

   :returns: A list of distances (minimum z-values relative to the ground).
   :rtype: list
