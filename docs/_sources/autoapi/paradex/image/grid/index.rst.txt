.. py:function:: make_image_grid(images: List[numpy.ndarray]) -> numpy.ndarray

   Tile a list of images into a nearly square grid (no labels).

   Odd-sized tiles are resized to the first image's shape. Returns a ``(1, 1, 3)``
   black pixel for an empty list.

   :param images: Images to tile, in order.
   :type images: list of numpy.ndarray

   :returns: The tiled grid image.
   :rtype: numpy.ndarray
